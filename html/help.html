Caminho do git

Repositório:

https://github.com/VicdaRocha/CatalogoFilmes.git

Fork

No link acima, faça um fork do repositório (area superior direita).

Isso criará uma cópia do repositório na sua conta do github.

Clone o seu repositorio

Depois de escolher um local para receber o repositório, abra o terminal e navegue até a pasta:

- cd caminho da pasta

- git clone https://github.com/meu-usuario/CatalogoFilmes.git

- cd CatalogoFilmes

Trazer branches do repositório remoto

- git fetch origin

Listar branches do repositório remoto

- git branch -r

Criar uma branch local a partir da sua branch remota

- git checkout -b nome-da-branch-local origin/NomeSobrenome

Quando estiver pronto:

- git status (lista arquivos alterados na branch)

Preparar mudanças para commit

- git add . (adiciona todas as mudanças à area de stage)

Caso não queira adicionar todas as mudanças, isso pode ser controlado de forma simples na aba de controle de versionamento do vs code, utilizando os botões com sinal de adição.

Commit

- git commit -m "digite uma mensagem para o commit(tem que estar entre aspas)"

Na aba de controle de versionamento: commit/confirmar

Publique sua branch

Cheque se sua branch está conectada à branch remota certa.

- git branch -vv

Se o resultado for seu NomeSobrenome:

- git push origin NomeSobrenome

Na aba de controle de versionamento: publish/publicar branch

Template

Deixei o arquivo "filme.html" de modelo, vcs podem só fazer uma cópia dele e preencher com o filme que escolherem, ou podem fazer uma página nova e subir do mesmo jeito.

Caso forem criar uma nova, criem um novo arquivo de CSS para cada arquivo de html, na pasta \styles

Salvar as imagens que usarem em \images
